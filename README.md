# UMIN000061709 公開データ / Public Dataset

## 概要 / Overview

本リポジトリは、臨床試験登録番号 **UMIN000061709** に対応する研究の公開データセットです。  
This repository contains the public dataset associated with the clinical trial registered as **UMIN000061709**.

8週間のウォーキング介入が循環血中 マイクロRNA（miRNA）発現プロファイルに与える影響を調べた前後比較研究のデータを収録しています。  
This is a pre-post intervention study examining the effects of an 8-week walking program on circulating micro RNA (miRNA) expression profiles.
